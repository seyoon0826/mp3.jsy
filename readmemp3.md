# 🎧 ALBUM PLAYER | MY PLAYLIST

이 프로젝트는 제가 좋아하는 아티스트들의 앨범을 한눈에 보고 음악을 감상할 수 있도록 만든 **웹 기반 앨범 플레이어**입니다.

## 🚀 프로젝트 소개
이번 프로젝트는 제가 **HTML을 처음 배우면서** 시작한 첫 번째 결과물입니다. 
단순한 텍스트 나열을 넘어, 실제 음악 앱처럼 세련된 UI를 구현하고 싶어 **AI(Gemini)**의 도움을 받아 CSS 디자인과 자바스크립트 기능을 강화했습니다. 페이지도 4개로 나누어서 홈과 뉴진스 페이지에서는 음악을, 소개에서는 소개글을, 블로그에서는 readme파일을 읽을 수 있다.

## ✨ 주요 기능
- **앨범 카드 인터페이스**: 각 앨범을 카드 형태로 구성하여 가독성을 높였습니다.
- **다크 모드 UI**: 스포티파이 스타일의 어두운 테마와 포인트 컬러(Green)를 적용했습니다.
- **중복 재생 방지**: 자바스크립트를 사용하여 한 곡을 재생하면 다른 곡은 자동으로 멈추는 스마트 플레이어 기능을 구현했습니다.
- **유튜브 연결**: 앨범 아트를 클릭하면 해당 앨범의 공식 영상으로 바로 이동합니다.
- **반응형 디자인**: 모바일에서도 앨범 정보가 깨지지 않고 최적화되어 보입니다.

## 🛠️ 사용 기술
- **HTML5**: 웹사이트의 전체적인 구조 설계
- **CSS3**: 다크 테마 디자인, 카드 레이아웃, 호버(Hover) 애니메이션 적용
- **JavaScript**: 오디오 중복 재생 방지 로직 구현# 🎧 ALBUM PLAYER | MY PLAYLIST.
